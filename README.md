# SCT_DS_2
TASK_2---Perform data cleaning and exploratory data analysis (EDA) on a dataset of your choice, such as the Titanic dataset from Kaggle. Explore the relationships between variables and identify patterns and trends in the data.

📌 Project Overview
This project focuses on analyzing the Titanic dataset using Python to understand the factors that influenced passenger survival. It involves data cleaning, handling missing values, exploratory data analysis (EDA), and data visualization to identify meaningful patterns and trends. By examining features such as age, gender, passenger class, and fare, the project provides insights into how these factors affected survival outcomes. This analysis helps build a strong foundation in data preprocessing, visualization, and real-world data analysis using Python.

🎯 Objectives
* Explore and understand the Titanic dataset.
* Clean and preprocess the data by handling missing values and duplicates.
* Perform Exploratory Data Analysis (EDA) to identify patterns and trends.
* Analyze the relationship between passenger attributes and survival.
* Visualize key insights using charts and graphs.
* Gain practical experience in data analysis using Python and Pandas.
* Develop a better understanding of real-world data preprocessing and visualization techniques.

*  📂 Dataset
* **PassengerId** – Unique identifier for each passenger
* **Survived** – Survival status (0 = Did Not Survive, 1 = Survived)
* **Pclass** – Passenger class (1st, 2nd, or 3rd)
* **Name** – Passenger's name
* **Sex** – Gender of the passenger
* **Age** – Age of the passenger
* **SibSp** – Number of siblings or spouses aboard
* **Parch** – Number of parents or children aboard
* **Ticket** – Ticket number
* **Fare** – Ticket fare
* **Cabin** – Cabin number (if available)
* **Embarked** – Port of embarkation (C = Cherbourg, Q = Queenstown, S = Southampton)


🛠️ Technologies Used
* **Python** – Programming language used for data analysis.
* **Google Colab** – Cloud-based platform for writing and executing Python code.
* **Pandas** – Used for data loading, cleaning, manipulation, and analysis.
* **NumPy** – Used for numerical computations and array operations.
* **Matplotlib** – Used to create charts and visualizations.
* **CSV Dataset** – Source of the Titanic passenger data used for analysis.
